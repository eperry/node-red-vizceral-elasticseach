# To run:  - NEED Nodejs installed

Checkout repo

npm install

./start

## open browser to 

http://localhost:1880   (Main screen)

http://localhost/admin     (Node-Red console)

http://localhost/toggle    (Toggel Simple, Complex or Live Elasticsearch data, and link to Raw Json Data)
